# Generator-crtrdg

> A generator for creating crtrdg.js games.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-crtrdg`
- Run: `yo crtrdg`
